# Weather-App-Kotlin-Hilt (almost the same as WeatherApp)

Android Kotlin application to read api from https://www.metaweather.com/ using Retrofit

Used Technologies:
- Retrofit (OkHTTP + GSON)
- Coroutines
- Flow
- Hilt
- MVVM
- Clean Architecture
- Timber
